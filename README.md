A space-themed social networking platform designed for seamless communication, networking, and collaboration.

## Features
-  User Profiles (Edit username and bio)
-  Broadcast Messages to Station
-  Like and Reply to Posts (with timestamps)
-  Add / Remove Contacts
-  Secure Direct Messaging (DMs)
-  Online / Offline Status

## How It Works 
1. Users sign up with email, password, username, and department  
2. Log in to access the dashboard  
3. Broadcast posts to the station  
4. Connect with crew members  
5. Start private conversations using secure messaging  

## Tech Stack
- HTML
- CSS
- JavaScript
- Database: Firebase

## Purpose
StellarNet is built to combine communication, networking, and personalization into one unified and visually engaging platform.
